# Rust REST API 

## Install

`git clone git@github.com:CrownBerry/rust-rest.git`

`cd rust-rest && rustup override set nightly`

`cargo update && cargo build`

`cargo run`

## Routes
### Person JSON object
Schema:
```
{
    "id": Option<int>,
    "name": String,
    "age": i32
}
```
Example:
```
{
    "name": "John Wick",
    "age": 47
}
```
